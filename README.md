# Онлайн магазин 

**Features**

В проекте использовались:
- webpack;
- препроцессор scss;
- адаптивная верстка;
- БЭМ
Для запуска проекта на локальном сервере используйте команду 'npm run dev', для сборки - 'npm run build'.
Есть возможность проверить проект на ошибки линтера командой 'npm run lint'.
